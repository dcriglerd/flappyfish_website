# How to Host Privacy Policy & Ads.txt on GitHub Pages (Free)

## Files to Upload
1. **index.html** - Privacy Policy page
2. **ads.txt** - Google AdMob verification

## Step-by-Step Guide (5 minutes)

### STEP 1: Go to GitHub
Open: https://github.com

If you don't have an account, create one (free).

---

### STEP 2: Create a New Repository

1. Click the **"+"** icon in the top right corner
2. Select **"New repository"**

```
┌─────────────────────────────────────────────────────────┐
│  Create a new repository                                 │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  Repository name:                                        │
│  [flappy-fish-privacy                              ]    │
│                                                          │
│  Description (optional):                                 │
│  [Privacy Policy for Flappy Fish game              ]    │
│                                                          │
│  ● Public   <-- SELECT THIS (required for free hosting) │
│  ○ Private                                               │
│                                                          │
│  ☑ Add a README file                                    │
│                                                          │
│                    [Create repository]                   │
└─────────────────────────────────────────────────────────┘
```

---

### STEP 3: Upload the Privacy Policy File

1. In your new repository, click **"Add file"** → **"Upload files"**

2. Drag and drop the `index.html` file (I'll provide this below)
   - OR click "choose your files" and select it

3. Scroll down and click **"Commit changes"**

---

### STEP 4: Enable GitHub Pages

1. Go to repository **Settings** (tab at the top)

2. Scroll down to **"Pages"** in the left sidebar

3. Configure:
```
┌─────────────────────────────────────────────────────────┐
│  GitHub Pages                                            │
├─────────────────────────────────────────────────────────┤
│                                                          │
│  Source:                                                 │
│  Branch: [main                    ▼]                    │
│  Folder: [/ (root)                ▼]                    │
│                                                          │
│                         [Save]                           │
│                                                          │
│  ✓ Your site is live at:                                │
│  https://YOUR-USERNAME.github.io/flappy-fish-privacy/   │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

4. Click **Save**

5. Wait 1-2 minutes for deployment

---

### STEP 5: Get Your Privacy Policy URL

Your privacy policy will be available at:

```
https://YOUR-USERNAME.github.io/flappy-fish-privacy/
```

Replace `YOUR-USERNAME` with your actual GitHub username.

**This is the URL you'll paste into Google Play Console!**

---

## Quick Test

After enabling GitHub Pages, visit your URL in a browser.
You should see the Flappy Fish Privacy Policy page.

If you see a 404 error, wait 2-3 minutes and refresh.
